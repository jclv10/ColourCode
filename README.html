En Local:

Requisitos

Node.js (con npm) Luego basta con hacer ""npm install"" en la carpeta del proyecto para instalar las dependencias.

Ejecutar app

npm run dev

Generar solución del nivel

Figura.figura_final es el archivo que contiene la solución del nivel actual. La función Figura.debugFinal() se ejecuta por consola y devuelve un array que representa la solución del nivel actual y ejecuta applyFinal() para actualizar la solución del nivel actual. La función Figura.applyFinal() actualiza la solución del nivel actual con la solución almacenada en Figura.figura_final.

En el servidor

Entrar dentro de la carpeta /app/pixi-project/ y lanzar el comando "npx vite"
